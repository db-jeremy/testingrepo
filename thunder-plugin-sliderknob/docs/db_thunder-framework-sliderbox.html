    Slider Knob

    Provides a content box which contains content which is panned left and right through a viewport by sliding a knob horizontally

     

    Usage:

      [tdr_framework_sliderknob notched="{true|false}"][slider]Content pane for first entry[/slider][slider]Content pane for second entry[/slider][/tdr_framework_sliderknob]
